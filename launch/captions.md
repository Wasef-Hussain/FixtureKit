# Screenshot Captions

---

**Caption 1 — Input pane**
> Paste any TypeScript interface or Zod schema. FixtureKit parses it instantly in the browser — no installs, no API calls.

---

**Caption 2 — Output pane**
> Get realistic, copy-ready fixture code in seconds. Field names like `email`, `name`, and `createdAt` are inferred semantically — not just filled with `"string"`.

---

**Caption 3 — Options bar / multi-fixture**
> Generate up to 5 distinct fixtures at once. Output is deterministic: the same schema always produces the same mocks, making fixtures safe to commit.
